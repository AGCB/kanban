# kanban

## install
- `cd` into the folder you'd like to download Kanban
- git clone the repo
- `cd` into the repo
- `npm install`
- `npm start`

## features
- add tasks
- move tasks across stages (icebox, progress, done)
- delete specific tasks

## template
- created using [create-react-app](https://github.com/facebook/create-react-app)
